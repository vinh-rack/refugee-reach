# Amazon Nova

Amazon Nova is a ***portfolio of foundation models and services*** that deliver frontier intelligence and industry-leading price performance. This is your chance to invent with freedom and build with Nova’s capabilities, including advanced reasoning, tool use, multimodal understanding, voice AI, and UI workflow automation. 
- [Nova 2 Lite](https://aws.amazon.com/nova/models/), a fast, cost-effective reasoning model for everyday tasks.
- [Nova 2 Sonic](https://aws.amazon.com/nova/models/), a speech-to-speech model for conversational AI.
- [Nova Act](https://aws.amazon.com/nova/act/) is a service to build and manage fleets of agents for automating production UI workflows with high reliability.
- [Nova multimodal embeddings](https://aws.amazon.com/nova/models/): A state-of-the-art multimodal embedding model.
